# Historical benchmark — preserved for context

This directory contains the **original benchmark** that this repository was
created to publish: a comparison of an HTML-scraping MCP server against
[BLZ](https://github.com/outfitter-dev/blz) for searching `docs.agno.com`.

It is preserved here for transparency and provenance. The current root of
the repository is a different artifact: the actual CLI (`agno-docs-pp-cli`)
that the benchmark implied should exist.

If you're looking for the CLI, see the [root `README.md`](../README.md).
If you want the benchmark methodology and numbers, the original files are
in this folder and in `_historical/`.
